# TuPrimeraPagina-Albanese

Proyecto final para el curso Python Flex – Coderhouse

Proyecto de Ferretería desarrollado en Django.

Este proyecto permite gestionar los productos de una ferretería, junto con sus categorías y proveedores. Además, cuenta con un sistema de login para empleados y diferentes vistas para mostrar la información de forma ordenada y clara.

El sitio fue desarrollado utilizando las siguientes tecnologías:

Python 3.12.10

Django 5.0

Bootstrap 5

La página cuenta con una vista de inicio donde se pueden ver y buscar los productos disponibles. Desde esta misma página o desde la barra de navegación se puede acceder a la sección Acerca de nosotros, donde se muestra información general sobre la ferretería.

Desde el navbar también es posible ingresar como empleado mediante el sistema de login. Una vez autenticado, el usuario puede acceder a funcionalidades adicionales como:

Agregar productos

Ver y administrar categorías

Ver y administrar proveedores

Editar y eliminar registros existentes

El proyecto incluye un sistema de usuarios con registro, login y logout, así como una vista de perfil donde se muestran y pueden editarse los datos del usuario.

Además, se implementa un panel de administración de Django donde se encuentran registrados todos los modelos creados en el proyecto.

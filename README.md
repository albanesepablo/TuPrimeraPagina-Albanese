# TuPrimeraPagina-Albanese
Proyecto para el curso python flex de coderhouse

Proyecto de Ferretería en Django  

Este proyecto permite gestionar productos, categorías y proveedores de una ferretería, login para empleados, y vistas para mostrar información.

Python 3.12.10
Django 5.0
Bootstrap 5

La pagina tiene un inicio donde se puede ver y buscar productos, como tambien un boton acerca de nosotros para ver informacion de la ferreteria, desde el body o navbar, tambien desde el navbar podremos ingresar como empleados, que nos dejara usar otras funcionalidades como agregar productos, agregar y ver categorias, agregar y ver proveedores.
